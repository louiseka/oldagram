## What is Oldagram?
**Oldagram** is a creative recreation of **Instagram**, reimagined as if it were used by **historical figures**.

The application displays posts generated from a dataset, allowing for a dynamic feed where content can easily be expanded by adding more data.

This project focuses on working with datasets, dynamic rendering, and JavaScript-driven UI logic.

## Features
- Instagram-style feed using historical figures
- Dynamically rendered posts from a dataset
- Scalable structure for adding additional content
- Clean and familiar social media-inspired UI
